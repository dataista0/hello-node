#Blog with node, express and mongodb

##Install express:
[Express] [2] is a web application framework for node.
```
sudo npm install express -g
express [yes]
cd . & npm install
```

Check for failures in dependencies: 
```
npm ls
```

Run this and check http://localhost:3000
```
node app
```
